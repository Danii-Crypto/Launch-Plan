How to Use the Launch Plan Management Dashboard

Getting Started

  1. Use the navigation tabs at the top to switch between different sections
  2. The Dashboard tab shows an overview of all your launch plans
  3. Use the search bar at the top right to quickly find specific clients
  4. All data is automatically saved to your browser's local storage

Creating New Launch Plans

  1. Go to the Launch Plan tab
  2. Click "Create New Launch Plan"
  3. Select your product type: Trading, Connect, or SAFE Custody
  4. Fill in the client information (name, account number, dates, AE, CS)
  5. Click "Create Launch Plan" to generate the plan with product-specific tasks
  6. The system will automatically create all relevant tasks based on your product selection

Managing Clients and Tasks

  1. Select a client from the dropdown in the Launch Plan tab
  2. Edit client information in the Project Overview section
  3. Click "Save Client Info" to save changes and sync all task dates
  4. Use the Task Management table to update task statuses
  5. Change task status by clicking the dropdown in the Status column
  6. Add new tasks using the "Add Task" button
  7. Use filters to view specific tasks (environment, priority, status, owner)

Client Management

  1. Go to the Manage Clients tab to see all clients
  2. Use "Show Archived" to toggle between active and archived clients
  3. Use "Show Completed" to view only completed launch plans
  4. Click the archive icon to archive/unarchive clients
  5. Click the edit icon to jump to a client's launch plan
  6. Use "Export All Launch Plans" to download comprehensive data

Reports and Analytics

  1. Go to the Reports tab for detailed analytics
  2. Use the filter options to narrow down your data view
  3. Apply filters by date range, AE, product type, or status
  4. Review the Executive Summary for key performance indicators
  5. Analyze the charts for visual insights on progress and performance
  6. Use "Export Report as PDF" to generate professional reports
  7. The detailed analytics table shows comprehensive client data

Search and Navigation

  1. Use the search bar at the top right to find clients quickly
  2. Search by client name, account number, AE name, or CS name
  3. Click on search results to jump directly to that client's launch plan
  4. The search works across all client data and is case-insensitive

Date Synchronization

  1. When you change project start or launch dates, all task dates can be synchronized
  2. After updating dates, click "Save Client Info" to sync all tasks
  3. The system maintains task duration and sequence relationships
  4. This works for all product types: Trading, Connect, and SAFE Custody

Tips and Best Practices

  • Regularly update task statuses to keep progress tracking accurate
  • Use the archive feature to organize completed or inactive launch plans
  • Export data regularly for backup and reporting purposes
  • Use filters to focus on specific priorities or overdue tasks
  • Review the analytics regularly to identify trends and improvements
  • The system automatically saves all changes to your browser
